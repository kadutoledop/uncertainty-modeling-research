# Uncertainty Modeling Research

This repository contains my research work on mathematical modeling and data science under uncertainty, with a focus on fuzzy systems, dynamical systems, and real-world data applications.

## 🔬 Research Topics
- Fuzzy data modeling
- Uncertainty quantification
- Dynamical systems
- Machine learning with noisy/imprecise data

## 📄 Publications

### 1. Interpolation with Fuzzy Input and Output and Spline Function
- Status: Published
- Description:

### 2. Qualitative Study of Epidemiological Models
