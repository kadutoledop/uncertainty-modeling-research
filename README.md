# Pesquisa em Modelagem sob Incerteza

Este repositório reúne meus trabalhos de pesquisa em modelagem matemática e ciência de dados sob incerteza, com foco em sistemas fuzzy, sistemas dinâmicos e aplicações em dados do mundo real.

## 🔬 Temas de Pesquisa
- Modelagem de dados fuzzy
- Quantificação de incerteza
- Sistemas dinâmicos
- Machine Learning com dados ruidosos/imprecisos
